Airtime Web Radio [full-time employee, 2010 - present]

http://en.wikipedia.org/wiki/Airtime
https://github.com/sourcefabric/Airtime
http://www.sourcefabric.org/en/airtime/

I am currently the team lead of this project. I lead code-review with team members, as well as assign tickets, plan system architecture, as well as write code. Airtime runs on Debian (and CentOS unofficially) based systems, and has gained quite a community following.

Some users create their own videos, here is one example:
http://www.youtube.com/watch?v=TqHs_jSYVOU

We have some fairly popular sites reviewing us as well:
http://www.omgubuntu.co.uk/2011/10/open-source-radio-station-management-software-airtime-sees-release

I also recently wrote a few blog posts about getting Airtime running on my Raspberry Pi: http://www.sourcefabric.org/en/community/blog/1490/Airtime-and-the-Raspberry-Pi-part-1-Experiments-in-installation.htm http://www.sourcefabric.org/en/community/blog/1574/Airtime-and-Raspberry-Pi-part-2-planning-for-peak-performance.htm

The main components Airtime uses is Apache, PHP 5.3 + Zend MVC, Postgresql and RabbitMQ.

A SaaS solution for Airtime is also available, and I am partially responsible for maintaining the server environment. Our SaaS environment uses nginx as a proxy to Apache - nginx is responsible for handling any DDoS attacks and Apache is responsible for serving static content and proxying to php-fpm.

http://platinumaxis.com/
https://itunes.apple.com/ca/app/platinumaxis/id584318980 (iPhone app)
https://play.google.com/store/apps/details?id=platinumaxis.android (Android app)

An app written in my spare time with some friends that show users which clubs are the hottest in Toronto. How "hot" a club is, is based on a variety of metrics such as social media (twitter and many others), as well as user's own ratings from within the app. All of this is handled in the backend and powered by several Amazon EC2 instances. PHP, MongoDB and Python are used in the backend - I was solely responsible for the backend. I am also responsible for a major part of the Android app.
